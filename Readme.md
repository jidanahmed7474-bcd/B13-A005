1️⃣ What is the difference between var, let, and const?

Answer: 
1) var:
Function-scoped (accessible within the function where it is declared).
Can be redeclared and reassigned.
It is hoisted and initialized with undefined.

2) let:
Block-scoped (only accessible inside { } where it is declared).
Cannot be redeclared in the same scope.
Can be reassigned.

3) const:
Block-scoped like let.
Cannot be redeclared.
Cannot be reassigned after initialization.

2️⃣ What is the spread operator (...)?

Answer: The spread operator (...) takes items from an array or object and spreads them out into individual elements.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer: 
Use map() to change data.
Use filter() to select data.
Use forEach() to run code on each item.

4️⃣ What is an arrow function?

Answer:
An arrow function is a shorter way to write a function using =>.

5️⃣ What are template literals?

Answer:
Template literals are strings written with backticks (`) that allow embedding variables and expressions using ${}.
